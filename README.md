# Persons in Crisis Calls and Police Apprehension

## Abstract
The Toronto Police Service is responsible for attending to high volume of Person in Crisis calls. There is much debate over whether or not they are the appropriate group to be dealing with situations involving mental illness and substance use. This paper provides evidence of a link between high volumes of calls and low proportions of apprehensions made across police divisions, which could indicate that over tasking police officers could lead to a mishandling of mental health crises.

## File Structure

The repo is structured as:

-   `data/raw_data` contains the raw Persons in Crisis Calls for Service Attended dataset as obtained from Open Data Toronto.
-   `data/analysis_data` contains the cleaned dataset that was constructed.
-   `paper` contains the files used to generate the paper, including the Quarto document and reference bibliography file, as well as the PDF of the paper. 
-   `scripts` contains the R scripts used to simulate, download and clean data.
